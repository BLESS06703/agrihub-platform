# AgriHub Malawi - Brand Identity & Design System
# Version 1.0

## 1. Brand Colors

### Primary Palette

| Purpose | Name | Hex |
|---------|------|-----|
| Primary | Forest Green | #2E7D32 |
| Secondary | Emerald Green | #43A047 |
| Accent | Harvest Gold | #F9A825 |
| Background | Off White | #F8FAF5 |
| Surface | White | #FFFFFF |
| Primary Text | Charcoal | #263238 |

### Rationale
- Green represents farming, growth, and nature
- Gold symbolizes harvests and prosperity
- Off-white provides a warm, natural background
- Charcoal ensures comfortable readability

## 5. Typography

- Primary Font: System default (Roboto on Android)
- Headings: Semi-bold
- Body: Regular
- Captions: Light
- Monospace: For data, numbers, codes

## 6. Iconography

Style: Outlined icons with rounded corners

Categories:
- Crops
- Livestock
- Equipment
- Inventory
- Finance
- Reports
- AI
- Marketplace

Recommended set: Material Symbols Rounded

## 7. Design Principles

1. Clean and professional - enterprise SaaS aesthetic
2. Agriculture-connected - colors reflect farming heritage
3. Accessible - WCAG 2.1 AA contrast compliance
4. Consistent - unified experience across Android and Web
5. Minimalist - no decorative emojis in UI or documentation
6. Trustworthy - inspires confidence for financial and farm data

## 8. UI Component Specifications

### Buttons
- Primary: #2E7D32 background, white text
- Secondary: #2E7D32 border, transparent background
- Disabled: #BDBDBD background
- Border radius: 8dp
- Minimum touch target: 48dp

### Cards
- Background: #FFFFFF (light), #2A2A2A (dark)
- Elevation: 2dp
- Border radius: 12dp
- Padding: 16dp

### Input Fields
- Border: 1dp #BDBDBD
- Focus border: 2dp #2E7D32
- Border radius: 8dp
- Error: 2dp #D32F2F

### Navigation Bar
- Background: #FFFFFF
- Selected icon: #2E7D32
- Unselected icon: #757575
