## Tech Stack

- Backend: Kotlin, Ktor, Exposed ORM
- Database: PostgreSQL 16 with PostGIS
- Cache: Redis | Storage: MinIO
- Auth: JWT + Argon2id
- Mobile: Kotlin, Jetpack Compose, Room

## Key Features

- AI Command Center
- Agricultural Intelligence Center
- Multi-tenant isolation
- Offline-first mobile sync
- Marketplace with digital contracts
- Multi-language support
