# AgriHub Platform - Component Catalog

## Buttons
- Primary: Forest Green bg, white text, 8dp radius
- Secondary: Forest Green border, transparent bg
- Danger: Error Red bg, white text
- Disabled: Grey bg, light text
- Size: height 48dp minimum

## Cards
- Background: White (#FFFFFF)
- Elevation: 2dp
- Radius: 12dp
- Padding: 16dp internal

## Input Fields
- Border: 1dp Grey (#BDBDBD)
- Focus: 2dp Forest Green
- Error: 2dp Error Red
- Radius: 8dp
- Label: Floating label animation

## Data Tables
- Header: Slate Grey, uppercase, 12sp
- Row: Charcoal, 14sp
- Border: Off White dividers
- Hover: Light green bg

## Navigation
- Bottom Bar (Mobile): 5 items, Forest Green active
- Sidebar (Web): 250px, Forest Green bg, white text
- Breadcrumb: Slash separated, Forest Green links

## Charts
- Line: For trends (yield, prices)
- Bar: For comparisons (crops, fields)
- Pie: For distributions (expenses, income sources)
- Area: For cumulative data

## States
Every component supports:
- Default
- Hover (web)
- Pressed
- Disabled
- Loading (skeleton)
- Error
- Empty
