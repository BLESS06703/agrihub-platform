# AgriHub Platform - Brand Manifest

## Logo
Wordmark: "AgriHub" in Forest Green (#2E7D32)
Tagline: "Smart Farming. Better Harvests."
Icon: Abstract leaf + hub node combination

## Typography
- Display: System Bold (titles, large numbers)
- Heading: System SemiBold (section headers)
- Body: System Regular (content, descriptions)
- Caption: System Light (metadata, timestamps)
- Monospace: JetBrains Mono (data, numbers, codes)

## Spacing Scale (dp)
4, 8, 12, 16, 20, 24, 32, 48, 64

## Corner Radius (dp)
- Buttons: 8
- Cards: 12
- Dialogs: 16
- Input fields: 8
- Chips: 20

## Elevation
- Surface: 0dp
- Card: 2dp
- Dialog: 8dp
- Navigation: 4dp

## Touch Targets
- Minimum: 48dp x 48dp
- Ideal: 56dp for primary actions
